# 解法1：组合数

走到终点，需要向右走m-1步，向下走n-1步，根据高中黑板上排列组合，你还不舍得解开吗？问题就演变成从m-1 + n - 1 = m + n - 2的总步数中，挑出n-1步向下走，有多少种组合？

Cm−1+n−1n−1C_{m-1+n-1}^{n-1} C​m−1+n−1​n−1​​

或者

Cm−1+n−1m−1C_{m-1+n-1}^{m-1} C​m−1+n−1​m−1​​

组合数公式：

Cnm=PnmPm=n!m!(n−m)!,Cn0=1C_n^m = {P_n^m \over P_m} = {n! \over m!(n-m)!}, C_n^0=1 C​n​m​​=​P​m​​​​P​n​m​​​​=​m!(n−m)!​​n!​​,C​n​0​​=1

var uniquePaths = function(m, n) {
  function cal(n) {
    let sum = 1;
    for(let i = 1; i <= n; i++) {
      sum *= i
    }
    return sum;
  }
  let max = Math.max(m, n);
  let min = Math.min(m, n);
  return cal(max - 1 + min - 1) / cal(min - 1) / cal(max - 1)
};

# 解法2：动态规划

想象一下如果只差一步就可以能走到重点d[i][j], 则当前位置可能是在终点上一格d[i-1][j]， 或者终点的前一格d[i][j-1], 那到终点的走法数就是

d[i][j]=d[i−1][j]+d[i][j−1]d[i][j] = d[i-1][j] + d[i][j-1] d[i][j]=d[i−1][j]+d[i][j−1]

以上就是动态方程

再考虑边界情况

d[i][j]=1,i=0∣∣j=0d[i][j] = 1, i=0 || j=0 d[i][j]=1,i=0∣∣j=0

var uniquePaths = function(m, n) {
  let result = Array.from({length: m}).map(() => [])
  for(let i = 0; i < n; i++) {
      result[0][i] = 1;
  } 
  for(let i = 0; i < m; i++) {
      result[i][0] = 1;
  }
  for(let i = 1; i < m; i++) {
      for(let j = 1; j < n; j++) {
          result[i][j] = result[i-1][j] + result[i][j-1];
      }
  }
  return result[m-1][n-1]
}
